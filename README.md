


### Dependencies
* Python
  * version >= 3.8
* Jira library for Python
  * requests >= 1.2.0
  * requests-oauthlib >= 1.3.0
  * requests-kerberos 
  * ipython
  * filemagic
* Django framework

### Installation 
* Install Python version 3.8 or more
  * https://www.python.org/downloads/
* Install Libraries for python

      pip install -r requeriments.txt
